Course Description

An introduction to the design of data-intensive, reliable, scalable, and maintainable systems. Includes an introduction to current and relevant tools, technologies, design principles, and frameworks.

Course Rationale

This course involves an overview the design and implementation of data intensive systems covering common approaches to processing big data at rest and data in motion.

Course Outcomes

After successful completion of this course, students should be able to:
  1. Describe the characteristics of data-intensive systems.
  2. Describe key sources of big data.
  3. Describe the V's of big data and identify what are (and are not) big data problems.
  4. Describe the Hadoop framework, HDFS file system, and MapReduce programming model.
  5. Design and execute a process using Hadoop.
  6. Design and implement MapReduce solutions.
  7. Describe distributed systems, their challenges and benefits, and apply concepts of good design to increasingly complex systems.
  8. Describe different ways to represent, store, manage, and process data.
  9. Describe different architectures and their strengths, weaknesses, and application areas.
  10. Describe current and relevant tools for data-intensive systems.
  11. Implement big data solutions using standard algorithms.
  12. implement data intensive systems using relational and nonrelational databases.
  13. Process large amounts of data in batch mode and streaming mode.

Course Outline and Expectations

The course is divided into an initial overview (Module 0) and seven content modules.
Module Content
0 Course overview, policies, and information
1 Big data: sources, Linux, problems and opportunities
2 Big data algorithms and tools, Hadoop, HDFS, MapReduce, and ecosystem
3 Text processing, regular expressions, MapReduce Projects
4 CDH, Lambda, Impala, HBase, Kafka, Spark, Streaming data
5 Kafka, Spark projects
6 Project demos
7 Project demos, course reflections
